## Phase 1: Isolated Build Foundation
Goal: Create the project-owned interpreter, output ownership, and reproducible dependency inputs.

Definition of Done: `.venv/` and `dist/` are ignored, the venv uses Python 3.13, and the build dependency lock is generated from reviewed inputs.

Tasks:
- [x] T001 [Config] Record the environment and artifact ownership model
  - DoD: `.env`, `.gitignore`, and scope docs define root `.venv` and `dist` without tracking either directory.
- [ ] T002 [Config] Generate the reviewed build dependency lock
  - DoD: `requirements/build.lock` is generated by `uv pip compile --generate-hashes` from `requirements/build.in`.
- [ ] T003 [QA] Validate the isolated interpreter
  - DoD: `.venv/bin/python` exists, is a venv, and can run `pip --version` after bootstrap.

Checkpoint: The venv and lock are available before the build script starts modifying package state.
